## 作成 / 編集
- [ ] 「+ 新規作成」でモーダルが開く。名前・対象エージェント・初回メッセージ・スケジュールを入力できる。
- [ ] スケジュール: プリセット（毎日/毎週/毎月/カスタム）切替で cron が更新され、次回3回プレビューが出る。
- [ ] DST リスク時（夏時間地域 × 1〜3時台）に warn 注記が強調される。
- [ ] 名前 or メッセージ未入力 / cron 無効のとき「作成」ボタンが無効。
- [ ] 作成後、一覧に追加され次回実行予定（API の upcoming_runs_at）が表示される。
- [ ] 既存行の「編集」で現在値がプリフィルされ、cron 変更を保存できる（部分更新）。

## 操作
- [ ] 「今すぐ実行」でトーストが出る（「履歴を開く →」付き）。手動 run が記録される。
- [ ] 一時停止/再開トグルで status が active ⇄ paused に切り替わる。paused 行は一時停止バナー。
- [ ] 「アーカイブ」で確認ダイアログ（不可逆の明示）→ 確定で一覧から消える。

## 実行履歴
- [ ] 行の「実行履歴 →」で breadcrumb サブビューに入る。サマリ（回数/成功/失敗）+ 「すべて/失敗のみ」フィルタ。
- [ ] 各 run 行に 時刻 / 成否 / エラー種別 / セッション ID が出る。「← 一覧へ」で戻れる。

## 実機の前提
- [ ] 対象エージェントが kintone MCP を使える環境（bootstrap environment）で動く。
- [ ] 通知/書き込みは初回メッセージ + MCP ツールでエージェント自身が行う想定。
